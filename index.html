Frequently Asked Questions

Need help understanding how SenseGrid works? Here are answers to some common questions about device setup, monitoring, and more.

What devices are supported by SenseGrid?

SenseGrid supports IR sensors, K-type temperature sensors, current sensors, and other industrial monitoring devices via ESP32. Custom sensors can also be integrated with minimal configuration.

Do I need internet to use the SenseGrid system?

Yes. SenseGrid uses Wi-Fi or GSM (via SIM module) for real-time data transmission. An internet connection is required for accessing the dashboard and alerts remotely.

Can I monitor multiple machines at once?

Absolutely. You can connect and monitor multiple machines across different locations using the same dashboard. Each device has a unique ID for organized tracking.

How are alerts and notifications handled?

Thresholds can be set for each sensor. When a reading exceeds the limit, you’ll receive instant mobile or email alerts via our ThingsBoard integration or custom logic rules.

Can I access the system on mobile?

Yes, our system includes a mobile-friendly dashboard so you can view real-time data, receive alerts, and manage devices directly from your phone or tablet.

Is support available if I run into issues?

We offer email and chat support for all plans. Enterprise users also receive dedicated phone support and custom integration assistance.
